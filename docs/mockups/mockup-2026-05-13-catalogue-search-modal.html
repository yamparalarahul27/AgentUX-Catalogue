Implementation notes
What's already on the table
R1 — ShortcutBoth ⌘K and Option+Space. ⌘K is the safer of the two — Option+Space can conflict with macOS system shortcuts in some configs.
R2 — ToolbarInline .catalogue-search pill removed from desktop, search icon added in toolbar-right alongside Share + Bookmarks.
R3 — MobileTapping the inline search bar opens the modal full-screen with the same categories.
R4 — CategoriesGroups, Flows, Screenshots (v1). Annotations / labels can be a 4th later.
R5 — Limits5 rows per category before "Show all N" expands. Modal max-height ~460px, scrolls inside.
R7 — MatchingWhitespace-tokenized fuzzy match. Each token must hit somewhere in the haystack. Case-insensitive.
