What's new vs X Posts cards: 16:9 thumbnail with a play overlay + YouTube source pill + duration badge. Title is primary text (X uses tweet excerpt). Secondary line is channel name + view count + posted date.

Click behaviour: opens the lightbox modal shown below — backdrop blur over the catalogue, centered 16:9 player, close (✕) top-right, click backdrop or press ESC to dismiss. Iframe unloads on close so playback stops + memory is released.

Schema impact: single new column kind: 'x' | 'youtube' on catalogue_video_references (default 'x'). All existing tags / comments / soft-delete / RLS plumbing works unchanged.

Metadata source: YouTube's free oEmbed API (youtube.com/oembed?url=...&format=json) via a new fetch-youtube-metadata Edge Function. No API key. Returns title + author_name + thumbnail_url.
